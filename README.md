# Keyboard to MIDI

Aplicación web que convierte el teclado del computador en un teclado MIDI, cubriendo todo el rango entre **C2 y C6**.

## Cómo usar

1. Abre `index.html` en Chrome/Edge (Web MIDI).
2. Selecciona una salida MIDI.
3. Toca este layout base (2 octavas completas + C):
   - `A W S E D F T G Y H U J K O L P ; ' ] \\ Z X C V B`
4. Mantén `Shift` para tocar la capa alta (mismas teclas +2 octavas).
5. Ajusta la velocidad con el deslizador.

## Rango

- Capa normal: **C2–C4**
- Capa con `Shift`: **C4–C6**
- Cobertura total combinada: **C2–C6** sin cambiar de registro con botones.
