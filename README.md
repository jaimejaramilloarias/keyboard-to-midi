# Keyboard to MIDI

Aplicación web simple que convierte el teclado del computador en un teclado MIDI para tocar instrumentos virtuales o dispositivos MIDI internos.

## Cómo usar

1. Abre `index.html` en Chrome/Edge (Web MIDI).
2. Selecciona una salida MIDI.
3. Usa las teclas `A W S E D F T G Y H U J K` para tocar.
4. Ajusta octava y velocidad según prefieras.

## Notas

- Si no aparecen dispositivos, conecta uno virtual/físico y pulsa **Actualizar**.
- La app envía Note On/Off por canal 1.
